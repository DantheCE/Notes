# Notes Application

Full-stack application for managing notes with a Node.js backend and MongoDB database.

## Project Structure

- `Backend/` - Express.js REST API server

## Quick Start

Navigate to the Backend directory and follow the instructions in its README:

```bash
cd Backend
npm install
npm run dev
```

## Requirements

- Node.js v14+
- MongoDB Atlas account or local MongoDB instance

## Documentation

See the [Backend README](./Backend/README.md) for detailed API documentation and setup instructions.
